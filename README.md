# admin-angular4
  fork [ngx-admin](https://github.com/akveo/ngx-admin)  
  简化了ngx-admin删除了一些没必要的模块 集成了阿里的[ng-zorro-antd](https://github.com/NG-ZORRO/ng-zorro-antd)
   
### 新增功能
- 登录注册
- 用户管理
- 菜单管理

### 执行命令
```
node >6.9.0 

# install
cnpm i 

# 开发
npm start  

# 构建
npm run build:prod

# 生产
npm run server:prod


```
